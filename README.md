# .github — профиль и дефолты организации

Спец-репозиторий организации **QuaTrade**. Содержит:

- [`profile/README.md`](profile/README.md) — страница организации (рендерится на
  её профиле для участников).
- [`CONTRIBUTING.md`](CONTRIBUTING.md), [`SECURITY.md`](SECURITY.md),
  [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md) — org-уровневые community-health
  файлы, используемые как дефолт для репозиториев без собственных.

Прикладного кода нет. Репозитории организации приватные.
